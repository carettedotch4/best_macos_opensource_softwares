## 🚀 About the Project

Welcome to **Best macOS Open-Source Softwares**! 
The goal of this repository is to gather the most useful, secure, and well-maintained open-source tools tailored for Mac users. Whether you are a developer, a designer, or just an everyday user, you'll find great alternatives to paid software here.

---

## 💻 The List

| App Name | Description | Link |
|---|---|---|
| **macUSB** | Download. Flash. Boot. The all-in-one USB creator for Mac. | [View Repository](https://github.com/Kruszoneq/macUSB) |
| **Docky** | A customizable Dock replacement for macOS with widgets, app folders, and a window switcher. | [View Repository](https://github.com/josejuanqm/docky) |
| **Plumb** | A lightweight macOS window manager that auto-centers and auto-tiles your apps. | [View Repository](https://github.com/Lv-0/plumb) |
| **Capso** | Open-source screenshot and screen recording for macOS. The free, native alternative to CleanShot X. Built with Swift 6.0 and SwiftUI. | [View Repository](https://github.com/lzhgus/Capso) |

## 🤝 How to Contribute

Contributions are what make the open-source community such an amazing place to learn, inspire, and create. Any contributions or suggestions to improve this list are **greatly appreciated**!

If you know a great open-source macOS app that isn't listed here:

1. **Fork** the Project
2. Create your Feature Branch (`git checkout -b feature/AddAwesomeApp`)
3. Commit your Changes (`git commit -m 'Add [App Name] to the list'`)
4. Push to the Branch (`git push origin feature/AddAwesomeApp`)
5. Open a **Pull Request**

*Make sure the software you are recommending is actively maintained and truly open-source!*
